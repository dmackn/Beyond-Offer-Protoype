# BeyondOffers

## Requirements
- Node.js v18+
- Free [Mapbox](https://mapbox.com) account

## Setup

# 1. Clone and install
git clone repo
cd into repo
npm install

# 2. Create a .env file in the project root
touch .env

Add your Mapbox token to .env:
VITE_MAPBOX_TOKEN=pk.your_token_here

Get your token at mapbox.com → sign up → copy the default public token from your dashboard.

# 3. Run it
npm run dev

Open link provided in chrome
